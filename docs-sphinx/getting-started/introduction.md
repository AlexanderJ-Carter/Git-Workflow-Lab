# 项目简介

欢迎来到 Git Workflow Lab！

## 什么是 Git Workflow Lab？

Git Workflow Lab 是一个**开源的 Git 学习平台**，专注于"边看边做"的实践体验。

### 🎯 目标

- 降低 Git 学习曲线
- 提供真实操作环境
- 展示现代工程化实践

### ✨ 特色

| 特性 | 说明 |
|------|------|
| 📚 系统课程 | 17+ 课时，从基础到进阶 |
| 🐳 一键环境 | Docker Compose 启动完整实验环境 |
| 🎮 交互学习 | 命令练习场、技能测验、AI 助手 |
| 🚀 CI/CD 示例 | GitHub Actions 完整工作流 |
| 🌐 双模式运行 | 公开站点 + 本地环境 |

## 课程体系概览

### 阶段 0-1: 基础与同步

安装配置、终端基础、commit、push、pull

### 阶段 2: 分支与协作

分支管理、PR、冲突处理、rebase、SSH

### 阶段 3: 救火与恢复

cherry-pick、revert、reflog、stash

### 阶段 4-5: 发布与自动化

标签、Release、CI/CD、Secrets
