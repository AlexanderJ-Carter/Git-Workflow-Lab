## 项目简介

**Git Workflow Lab** 是一个面向 Git 学习与协作实践的教学仓库，聚焦"边看边做"的课程体验。

### 核心特性

- 📚 **系统化课程** - 17+ 课时，从基础到进阶
- 🐳 **Docker 实验环境** - Gitea + Web 终端一键启动
- 🚀 **CI/CD 示例** - GitHub Actions 完整工作流
- 🎯 **交互式学习** - 命令练习场、技能测验、AI 助手
