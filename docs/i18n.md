# Git Workflow Lab - i18n Configuration

## Internationalization (i18n) Support

This project supports multiple languages to make Git education accessible globally.

### Supported Languages

- 🇨🇳 **Chinese (Simplified)** - Primary language, fully supported
- 🇺🇸 **English** - Documentation and interface partially translated

## Translation Files

### Sphinx Documentation

Sphinx supports internationalization through its built-in i18n system. Translation files are located in:

```
docs-sphinx/locale/
├── en/
│   └── LC_MESSAGES/
│       ├── index.po
│       ├── getting-started.po
│       └── ...
└── zh_CN/
    └── LC_MESSAGES/
        └── ...
```

### Website Interface

The main website (`site/`) currently supports Chinese. English version planning is in progress.

## Building Translated Documentation

### Build English Documentation

```bash
# Generate POT files
cd docs-sphinx
make gettext

# Create translation for English
sphinx-intl update -p _build/gettext -l en

# Build English documentation
make html BUILDDIR=_build/en -e SPHINXOPTS="-D language='en'"
```

### Build Chinese Documentation

```bash
cd docs-sphinx
make html
```

## Contributing Translations

We welcome translation contributions! Here's how to help:

### 1. Translate Documentation

1. Fork the repository
2. Navigate to `docs-sphinx/locale/<language>/LC_MESSAGES/`
3. Edit `.po` files with your translations
4. Submit a pull request

### 2. Translate Website

1. Website translation files will be located in `site/i18n/`
2. Create or update JSON files for your language
3. Test locally
4. Submit a pull request

### 3. Translation Guidelines

- Use clear, simple language
- Maintain consistent terminology
- Keep Git command examples unchanged
- Preserve Markdown formatting
- Add context comments when needed

## Multi-Language Deployment

### GitHub Pages

The project deploys to GitHub Pages with language selection:

- Main site: https://alexanderj-carter.github.io/Git-Workflow-Lab/
- English docs: https://alexanderj-carter.github.io/Git-Workflow-Lab/en/

### Local Development

```bash
# Serve Chinese version (default)
make docs-serve

# Serve English version
cd docs-sphinx && make html BUILDDIR=_build/en -e SPHINXOPTS="-D language='en'"
cd _build/en && python -m http.server 8001
```
