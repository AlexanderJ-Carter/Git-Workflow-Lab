🚀 初始化与克隆

git init

在当前目录初始化 Git 仓库

git clone <url>

克隆远程仓库到本地

git config --global user.name "名称"

设置全局用户名

git config --global user.email "邮箱"

设置全局邮箱

📋 状态查看

git status

查看工作区状态 最常用

git status -s

简洁格式显示状态

git diff

查看未暂存的修改

git diff --cached

查看已暂存的修改

git diff <file>

查看指定文件的修改

💾 提交操作

git add <file>

添加文件到暂存区

git add .

添加所有修改（不含删除）

git add -A

添加所有修改（含删除） 推荐

git add -i

交互式添加

git commit -m "消息"

提交暂存区内容 最常用

git commit -am "消息"

添加并提交（仅限已跟踪文件）

git commit --amend -m "新消息"

修改最后一次提交

🌿 分支操作

git branch

列出本地分支

git branch -r

列出远程分支

git branch -a

列出所有分支

git branch <name>

创建新分支

git checkout -b <name>

创建并切换分支 常用

git checkout <branch>

切换到指定分支

git switch -c <name>

创建并切换（Git 2.23+）

git merge <branch>

合并指定分支到当前分支

git rebase <branch>

变基到指定分支 重要

git branch -d <name>

删除本地分支

git branch -D <name>

强制删除本地分支

☁️ 远程操作

git remote -v

查看远程仓库

git fetch

获取远程更新（不合并）

git pull

拉取并合并 最常用

git pull --rebase

使用 rebase 方式拉取

git push

推送到远程 最常用

git push -u origin <branch>

推送并设置上游分支

git push origin --delete <branch>

删除远程分支

📜 历史查看

git log

查看提交历史

git log --oneline

简洁格式 最常用

git log --graph --oneline --all

图形化显示分支

git log --pretty=format:"%h %s"

自定义格式显示

git show <commit>

查看指定提交详情

git blame <file>

查看文件每行最后修改

📦 暂存操作

git stash

暂存当前修改

git stash list

查看暂存列表

git stash pop

恢复并删除最近暂存

git stash apply

恢复但保留暂存

git stash drop

删除最近暂存

git stash clear

清空所有暂存

⚡ 高级操作

git reset HEAD~1

撤销最后一次提交（保留修改）

git reset --hard HEAD~1

撤销最后一次提交（丢弃修改）危险

git revert <commit>

安全撤销指定提交 重要

git reflog

查看操作历史（找回丢失的提交）重要

git cherry-pick <commit>

复制指定提交到当前分支

git rebase -i HEAD~n

交互式变基（修改最近 n 个提交）

git tag -a v1.0.0 -m "版本说明"

创建附注标签

git clean -fd

清理未跟踪文件
