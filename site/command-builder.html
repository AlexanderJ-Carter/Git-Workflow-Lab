1

生成 SSH Key

在终端中运行以下命令生成新的 SSH 密钥对：

ssh-keygen -t ed25519 -C "your_email@example.com"
2

添加到 SSH Agent

启动 SSH Agent 并添加私钥：

eval "$(ssh-agent -s)" ssh-add ~/.ssh/id_ed25519
3

添加公钥到 Git 服务

复制公钥内容并添加到 GitHub/GitLab 等平台：

cat ~/.ssh/id_ed25519.pub

Ed25519 算法被推荐使用，比 RSA 更安全且密钥更短。如果你的系统不支持，可以使用 rsa 算法：ssh-keygen -t rsa -C "your_email@example.com"
