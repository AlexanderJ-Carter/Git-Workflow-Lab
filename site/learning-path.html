推荐学习顺序

如果你是 Git 新手，建议按照以下顺序学习：

  1. 阶段 0-1 基础入门：理解 Git 的基本概念和最常用的命令
  2. 阶段 2 分支与协作：学会使用分支，理解团队协作流程
  3. 阶段 3 救火与恢复：掌握误操作后的恢复技能
  4. 阶段 4-5 CI/CD：了解自动化和工程化实践
