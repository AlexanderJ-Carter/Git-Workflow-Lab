❓ 常见问题

为什么工作台左侧看不到课程内容？

请尝试按 Ctrl+F5 强制刷新页面清除缓存。如果问题仍然存在，请检查浏览器控制台的错误信息。

Web 终端连接不上怎么办？

确保 Docker 容器正在运行。可以在终端执行 docker-compose ps 检查服务状态。

如何停止所有服务？

在项目根目录执行 docker-compose down 停止所有 Docker 服务。
